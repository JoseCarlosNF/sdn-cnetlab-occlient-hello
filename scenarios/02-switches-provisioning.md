Se tudo correu bem, você deve conseguir acessar a interface do ONOS no endereço
a seguir:

<http://localhost:8181/onos/ui>

As **credenciais padrão** são as mesmas do acesso via ssh:

```plain Credenciais do onos
User: onos
Password: rocks
```
